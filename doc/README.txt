here some Information is gathered fot the CUBEmixer.

Please see in: http://ambisonics.iem.at/xchange/products/cubemixer for actual information.


Content:

examples: some exampledata for implementation. This directory will grow over time.
olddocs: historic docs for helping converting old pieces for the CUBEmixer to new versions
reference_docs: some data about names and controllers, generated from CUBEmixer or documented.

BUGS.txt: an hint where you can find BUGS and FEATURE request
InstallationTutorial.pdf: a copy of the online version at http://ambisonics.iem.at/xchange/products/cubemixer/documentation
IEM_Report.pdf: A german manual, outdated but sometimes useful ;-)
